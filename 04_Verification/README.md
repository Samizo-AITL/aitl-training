# Competence Verification

This directory defines **how training outcomes are verified**.

Verification documents are used to:
- Confirm actual competence
- Ensure responsibility boundaries are understood
- Prevent unsafe delegation to AI or tools

Verification focuses on:
- Explainability
- Judgment
- Accountability

Results of verification are referenced in training records.
