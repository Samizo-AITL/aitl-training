# Improvement

This directory describes **continuous improvement and corrective-action concepts**
related to AITL education and training.

Documents here explain:
- How training feedback is reviewed
- How gaps in competence are addressed
- How lessons learned are reflected in training updates

This directory supports **ISO 9001 Clause 10 (Improvement)**.
